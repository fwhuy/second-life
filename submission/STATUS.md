## ⚠️ Known risk — ConvNeXt has no independent test number

ConvNeXt was trained with a single seed-42 train/val split and selected on that
val fold, so **98.30% is validation, not test.** This is stated honestly in the
paper.

- **Not fatal:** Swin-B has a genuine held-out test (97.61% / 97.82% TTA), so the
  project *does* implement train/val/test; the ablation + confusion matrix cover
  the analysis bands. Realistic band is 15–23, not the 0–8 "no test set" floor.
- **The cost:** Track 1 ranks peers on *test accuracy*; our cap-compliant model
  only has a validation number, and judges will ask "why no test?" (the Q&A sheet
  answers this honestly).
- **Optional fix (no retrain):** run ConvNeXt on an independent dataset it never
  trained on (TrashNet), deduplicated against its training corpus → a real
  cross-dataset test number + confusion matrix. Lower than 98.30% (different
  distribution) but genuine. **Team decision: do we add this?**
